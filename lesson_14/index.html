Тег input - блочно-рядкові

Тип text (за замовченням) - Однорядковий ввід даних

"input" name="form[]" type="text"

Тип name - ім'я поля (даних)

"input" name="form[]" type="text"

Тип value - значення поля (дані)

"input" name="form[]" value="це value" type="text"

Тип placeholder - підпис поля

"input" name="form[]" placeholder="Введіть своє і'мя" type="text"

Тип placeholder - підпис поля зникає

"input" name="form[]" placeholder="Введіть своє і'мя" type="text"

.form__input--focus::placeholder { color: #333; transition: opacity 0.8s; }

.form__input--focus:focus::placeholder { opacity: 0; }

Тип autocomplete - дозволяє (за замовченням) / забороняє (значення off) автозаповнення поля

"input" autocomplete="on" name="name" placeholder="Введіть своє і'мя" type="text"

Тип required - обо'язково для заповнення

"input" required autocomplete="off" name="name" placeholder="Введіть своє і'мя" type="text"

Тип tabindex - керує навігацією за допомогою TAB

"input" tabindex autocomplete="off" name="name" placeholder="Введіть своє і'мя" type="text"

Тип maxlength - максимальна кількість символів доступних для вводу

"input" maxlength="5" name="form[]" placeholder="Введіть своє і'мя" type="text"

Тип minlength - вимога до мінімальної кількості символів. працює в парі з required

"input" required minlength="5" name="form[]" placeholder="Введіть своє і'мя" type="text"

Тип disabled - поле перестає бути доступним для користування, данні передані не будуть

"input" required minlength="5" name="form[]" placeholder="Введіть своє і'мя" type="text"

Тип disabled - поле перестає бути доступним для користування, данні передані не будуть

"input" required minlength="5" name="form[]" placeholder="Введіть своє і'мя" type="text"

.form__input--stimul:disabled { opacity: 0.5; }

Тег label - рядковий

Атрибут for - вказуємо ID поля з яким хочемо з'вязати label

в label вказуємо for="назва псевдо класса" / в input вказуємо id="назва псевдо класса"

Тег input / Атрибут type - тип елементу вводу даних

type - Тип email - Однорядковий ввід поштової адреси

input class="form__input" required name="form[]" placeholder="Введіть свій email" type="email"

type - Тип hidden - Приховане поле. Використовується разом з value

input class="form__input" value="windows" name="form[]" type="hidden"

type - Тип tel - поле вводу телефонного номеру

input class="form__input" name="form[]" placeholder="Введіть свій номер телефону" type="tel"

type - Тип password - поле вводу паролю

input class="form__input" name="form[]" placeholder="Введіть свій пароль" type="password"

type - Тип number - поле вводу числа

input class="form__input" name="form[]" placeholder="Введіть число" type="number"

type - Тип color - поле вибору кольору

input class="form__input" name="form[]" value="#22d360" type="color"

type - Тип search - поле пошуку

input class="form__input" name="form[]" type="search"

type - Тип date - поле вибору дати

input class="form__input" name="form[]" type="date"

type - Тип range - поле вибору значення

input class="form__input form__input--range" value="0" name="form[]" type="range"

type - Тип checkbox - множинний вибір

input class="form__input" id="for-5" checked name="form[]" type="checkbox"

type - Тип checkbox - множинний вибір

input class="form__input" id="for-5" name="form[]" type="checkbox"

type - Тип radio - одиночний вибір

Варто вказувати значення атрибуту value

input checked id="model-01" name="model" value="form[]" type="radio"

Атрибут multiple - додавання групи файлів

input class="form__input--range" multiple accept=".jpg, .png" id="file" name="form[]" type="file"

Тег select - випадаючій список. Атрибут multiple.

Тег button - Кнопка

type="button" - кнопка нічого не робить

type="reset" - обнуляє (скидає) поля вводу у формі. Має знаходитить в середені тегу form.

type="submit" - (за замовченням) відправляє форму. Має знаходитить в середені тегу form.
